# Medical Study Suite

An AI-powered study tool for medical students, PA students, NP students, residents, and healthcare providers.

## Features
- 🩺 **Case & Quiz** — Board-style clinical vignettes and MCQs
- 📄 **Study Guide** — Turn raw notes into structured study guides
- 🧠 **DDx Coach** — Differential diagnosis coaching
- 🔬 **Deep Dive** — Full clinical rundown on any condition

## Deployment (Vercel)
This app deploys automatically via Vercel when connected to GitHub. No additional configuration needed.

## Disclaimer
All content is AI-generated for educational purposes only. Not a substitute for clinical judgment. Always verify with UpToDate, First Aid, PANCE Prep Pearls, or your program faculty.
